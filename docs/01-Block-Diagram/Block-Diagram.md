## Overview
This moisture sensor block diagram exists to show the basic construction of the sensing subsystem of the Garden Buddy product. The sensor is connected to the motor control board allowing irrigation to occur only at certain moisture levels. This subsystem will be connected to the motor board via an 8-pin ribbon connector.
